# Patinhas na Nuvem (PROJETO-2026) 🐾

Este projeto é um sistema de gerenciamento para ONGs de adoção de animais, construído com React + Vite.

## Tecnologias

- React
- Vite
- Lucide React (Ícones)
- Formspree (Integração de Formulários)

## Como Rodar

1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
