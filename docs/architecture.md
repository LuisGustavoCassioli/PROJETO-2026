# Arquitetura do Projeto 🏗️

## Estrutura de Pastas

Como engenheiro senior, recomendo a seguinte organização:

- `src/components`: Componentes reutilizáveis (UI/Shared).
- `src/pages`: Componentes de página (Views).
- `src/services`: Lógica de API e integrações externas (ex: Formspree, Supabase).
- `src/assets`: Imagens e arquivos estáticos compiláveis.
- `src/utils`: Funções auxiliares (formatadores, validadores).
- `src/hooks`: Hooks customizados para lógica compartilhada.

## Padrões de Design

- **Glassmorphism**: Uso de transparências e blur para profundidade.
- **Micro-interações**: Transições suaves e hover effects para feedback visual.
- **Responsividade**: Mobile-first approach em novos componentes.
