# Guia de Contribuição 🤝

Obrigado por seu interesse em contribuir com o **Patinhas na Nuvem**! Sua ajuda é essencial para melhorar a vida dos animais.

## Como começar?

1. **Encontre um problema**: Verifique as [Issues](https://github.com/LuisGustavoCassioli/PROJETO-2026/issues).
2. **Faça um Fork**: Crie sua própria cópia do repositório.
3. **Desenvolva**: Implemente sua solução ou melhoria.
4. **Teste**: Garanta que tudo continua funcionando corretamente.
5. **Pull Request**: Envie suas mudanças para revisão.

## Padrões de Código

- **Componentes**: Use componentes funcionais e hooks.
- **CSS**: Siga os tokens definidos em `index.css`.
- **Commits**: Use mensagens claras e objetivas (ex: `feat: add search history`).

## Estilo de Design

Lembre-se de manter a estética **Premium** do projeto:

- Layouts limpos.
- Gradientes suaves.
- Uso de `backdrop-filter: blur()`.
- Micro-animações de entrada e hover.
